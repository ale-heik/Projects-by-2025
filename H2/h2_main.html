Tehtävä 3 [4p]

Palautus

Koodi tulostaa ensin kaikki taskit tyylillä
"Task1 to execution..."
"Task2 to execution..."
"Task3 to execution..."

Tämän jälkeen tulee tulostus
Last code line executed!

viimeisenä tulee tulosteet
${str} finished! eli;
Task3 finished!
Task1 finished!
Task2 finished!

koska
const print = (str, taskTime) => {
setTimeout(function() {
console.log(`${str} finished!`)
}, taskTime)
}
odottaa tasktimen verran aikaa, jonka jälkeen tulosteet tulevat
tässä tapauskessa 500ms ja 2000ms
